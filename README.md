# Guide Planner Pro

Déploiement ultra simple sur GitHub + Vercel

## Fichiers
- `index.html`
- `vercel.json`

## Déploiement
1. Dézippe ce dossier.
2. Sur GitHub, ouvre ton repo `Luxairtours`.
3. Clique **Add file** → **Upload files**.
4. Glisse `index.html` et `vercel.json`.
5. Clique **Commit changes**.
6. Retourne sur Vercel.
7. Clique **Add New** → **Project**.
8. Choisis ton repo GitHub `Luxairtours`.
9. Clique **Deploy**.

## Important
Cette version est une web app statique.
Elle peut lire PDF, DOCX, TXT, CSV, images via des bibliothèques web chargées côté navigateur.
Les anciens fichiers `.doc` Word peuvent rester limités.
